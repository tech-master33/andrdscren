# andrdscren

[![Android CI](https://github.com/techmaster300/andrdscren/actions/workflows/android.yml/badge.svg)](https://github.com/techmaster300/andrdscren/actions/workflows/android.yml)

An Android application for screen reading and TTS management.

## Features
- Screen Reader Service
- TTS (Text-to-Speech) Management
- Modern UI using Jetpack Compose

## Building
This project uses Gradle. To build the project locally, run:
```bash
./gradlew assembleDebug
```

## CI/CD
GitHub Actions is configured to automatically build the project on every push to `main` or `master`. You can find the generated APKs in the "Actions" tab of the GitHub repository.
